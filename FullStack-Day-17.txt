âœ… 1. filter() Function
ðŸ”¹ Purpose:
filter() is used to filter elements from an iterable (list, tuple, etc.) based on a condition.

ðŸ”¹ Syntax:
filter(function, iterable)

Returns only elements where the function returns True.

ðŸ§  Example: Filter even numbers
nums = [1,2,3,4,5,6]

even_nums = list(filter(lambda x: x % 2 == 0, nums))
print(even_nums)   
# Output: [2, 4, 6]

ðŸ§  Example: Filter names with length > 5
names = ["ram", "sita", "lakshman", "bharat"]

long_names = list(filter(lambda x: len(x) > 5, names))
print(long_names)
# Output: ['lakshman', 'bharat']


âœ… 2. map() Function
ðŸ”¹ Purpose:
map() applies a function to every element of an iterable and returns a new iterable.

ðŸ”¹ Syntax:
map(function, iterable)

ðŸ§  Example: Square of each number
nums = [1, 2, 3, 4]

squares = list(map(lambda x: x*x, nums))
print(squares) 
# Output: [1, 4, 9, 16]

ðŸ§  Example: Convert list of strings to uppercase
words = ["python", "map", "function"]

upper_words = list(map(lambda w: w.upper(), words))
print(upper_words)
# Output: ['PYTHON', 'MAP', 'FUNCTION']


âœ… 3. reduce() Function
ðŸ”¹ Purpose:
reduce() is used to apply a function cumulatively to the items of an iterable.

Note: reduce() is in functools module.

ðŸ”¹ Syntax:
reduce(function, iterable)

ðŸ§  Example: Sum of all numbers (using reduce)
from functools import reduce

nums = [1,2,3,4,5]

result = reduce(lambda a, b: a + b, nums)
print(result)
# Output: 15


ðŸ§  Example: Find product of a list
from functools import reduce

nums = [2, 3, 4]

product = reduce(lambda a, b: a * b, nums)
print(product)
# Output: 24
